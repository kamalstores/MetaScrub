# MetaScrub

MetaScrub is a privacy-first, 100% client-side web application designed to strip hidden metadata such as GPS coordinates, device fingerprints, timestamps, and personal data from your photos and videos. Everything runs locally in your browser, ensuring no uploads and zero server interaction.

## Features
- Zero-Upload Privacy: Files never leave your device.
- WASM-Powered Speed: Fast processing for images and videos directly in your browser.
- Selective Metadata Control: See what data is hidden and easily toggle off sensitive fields.
- Batch Processing: Clean multiple files at once.
- Non-Blocking UI: Smooth performance during heavy processing.
- Offline Capability: Fully functional without an internet connection.
